4. Arama (Searching)

GET /products/_search
{
  "query": {
    "match": { "name": "Mouse" }
  }
}

5. Aggregations

GET /products/_search
{
  "size": 0,
  "aggs": {
    "avg_price": { "avg": { "field": "price" } }
  }
}
